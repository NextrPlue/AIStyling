# AIStyling

1. 최종 목표: 웹 서비스 개시

2. 프로세스  
  a. 사용자가 전신 사진을 업로드
    - 가이드가 있어야 함.  
    - 사진의 밝기, 채도도 중요(A4 용지?)  
  b. 얼굴형, 퍼스널컬러, 체형을 분석  
  c. 분석한 결과를 라벨링  
  d. 라벨링 데이터를 토대로 패션 추천  
    - 패션 또한 라벨링 되어 있어야 한다.(데이터 존재함)  
    - 라벨링 데이터 간의 최적의 조합도 존재 되어야 한다.  

3. 개발 언어
- 백엔드: Spring
- 프론트엔드: React
- 알고리즘: 파이썬
